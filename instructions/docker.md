# Χρήση του Docker

1. Εγκατάσταση του [Docker Desktop](https://www.docker.com/products/docker-desktop/)
2. `git clone https://github.com/vagos/os-exercises.git`
3. `cd os-exercises`
4. `docker build -t ubuntu-xv6 .`
5. `docker run --rm -it ubuntu-xv6`
